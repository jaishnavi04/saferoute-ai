# SafeRoute AI 🚀

## 📌 Overview
SafeRoute AI is an AI-powered navigation system that provides the safest routes instead of the fastest.

## 🚨 Problem
Existing navigation apps focus on speed, ignoring safety factors like crime, lighting, and crowd density.

## 💡 Solution
SafeRoute AI analyzes real-time data to recommend safer routes using risk scoring.

## ✨ Features
- Safe route recommendation
- AI-based risk scoring
- Real-time alerts
- SOS emergency support

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (FastAPI)
- APIs: Google Maps API
